# Tic Tac Toe

A classic 2-player Tic Tac Toe game built with HTML, CSS, and JavaScript — no frameworks, no dependencies.

🔗 **Live Demo:** https://bharat-sharma88.github.io/Tic-Tac-Toe-Game/

## Features

- Classic 3x3 grid, 2-player gameplay (X and O)
- Automatic win detection across all rows, columns, and diagonals
- Draw detection when the board fills with no winner
- Color-coded players (blue for X, purple for O)
- One-click restart without reloading the page
- Clean, responsive layout using `vmin` units

## Tech Used

- HTML5
- CSS3 (Flexbox layout, hover transitions)
- JavaScript (DOM manipulation, game state logic)

## How to Play

1. Player X goes first — click any empty box to place a mark
2. Turns alternate automatically between X and O
3. First to get three in a row (horizontally, vertically, or diagonally) wins
4. If all 9 boxes fill with no winner, it's a draw
5. Click **Restart** to reset the board and play again
